# Personality Engine — Unified Personality System for Repo-Native Agents

> *Every agent has a voice. This is how they find it.*

## What It Is

A unified system for defining, evolving, and sharing AI agent personalities. Personalities are not static prompts — they are living configurations that grow with the agent and adapt to context.

## Features

- 🎭 **Personality profiles**: Voice, tone, knowledge domain, communication style
- 🧬 **DNA-based**: Core personality is defined at creation, like cell DNA
- 🔄 **Contextual adaptation**: Personality shifts based on who is talking and what about
- 📚 **Personality sharing**: Fork a personality, customize it, share it back
- 🎨 **Multi-host**: Different personalities for different hosts (like LucidDreamer)

## Integration

- LucidDreamer: Multiple podcast hosts with distinct personalities
- DMLog: Dungeon Master personality with genre awareness
- FishingLog: Personality scripts (veteran, nervous, pure fisher)
- PersonalLog: Adapts personality to user's communication style over time

## Status

📝 Personality systems exist in luciddreamer-ai and fishinglog-ai. This repo unifies them.
